# 智能水环境监测无人船项目说明

## 一、项目背景

针对传统水质监测人工巡检效率低、固定检测点覆盖不足以及数据智能分析能力弱等问题，本项目设计了一套智能水环境监测无人船。

无人船能够自主航行至指定区域，对水环境进行移动式、多参数、智能化监测。

## 二、项目简介

系统采用"RDK X5 + STM32F4"双层控制架构：

-   RDK
    X5负责高层智能任务，包括ROS2机器人系统、自主导航、SLAM建图、视觉识别和AI分析。
-   STM32F4负责底层实时控制，包括双电机控制、水质采集、遥控处理和安全保护。

通过船载端、通信链路以及云端平台，实现：

**自主航行 → 水质采集 → 智能分析 → 云端监管 → 异常报警**

完整闭环。

## 三、主要功能

### 1. 自主巡航

利用激光雷达、摄像头和IMU/GPS融合定位：

-   环境建图
-   路径规划
-   动态避障
-   定点悬停
-   一键返航

### 2. 水质检测

支持以下指标：

-   水温
-   pH
-   浊度
-   TDS

传感器数据经过滤波、校准后上传至上位机分析。

### 3. AI异常检测

利用历史水质数据建立动态变化模型，实现：

-   自适应阈值调整
-   异常污染识别
-   多参数联合分析

降低传统固定阈值造成的误报警。

### 4. 云端管理

提供：

-   船位实时显示
-   水质曲线展示
-   历史数据查询
-   异常告警推送
-   数据报表导出

## 五、软件架构

### RDK X5端

运行：

-   Ubuntu 22.04
-   ROS2 Humble

主要模块：

  模块            功能
  --------------- ------------
  Cartographer    SLAM建图
  Navigation2     自主导航
  YOLOv26         视觉检测
  水质分析模块    AI异常判断
  STM32通信模块   数据交换

### STM32端

负责：

-   双电机差速控制
-   PWM输出
-   水质传感器读取
-   2.4G遥控
-   串口通信

## 七、项目亮点

1.  双层异构控制架构，提高系统稳定性。
2.  ROS2机器人框架，实现自主移动能力。
3.  边缘AI实现智能水质分析。
4.  端-边-云协同，提高无人化水平。
5.  支持复杂水域长期监测。
